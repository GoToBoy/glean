# Sprint Contract Template

```md
# Sprint Contract

## Goal

One sentence describing what this sprint must achieve.

## Scope

- Files or modules expected to change
- Explicit exclusions

## Done Means

- User-visible or system-visible outcomes
- Exact checks or tests that must pass

## Risks

- Known ambiguity
- Migration or compatibility concerns

## Evaluator Focus

- Specific behaviors most likely to regress
```
